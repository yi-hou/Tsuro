# xobs
1. In terminal, enter **./xobs**
1. Type placements, one line for one placement
1. Hit **ctrl+D** and **enter**
- e.g. 

    ./xobs
    
    [["white",22,0, 0, 0,"H"],2,3,22]
 
# xref
1. In terminal, enter **./xref**
1. Type list of players
1. Hit **ctrl+D** and **enter**
- e.g. 

    ./xref
    
    ["jack","alice","bob"]
